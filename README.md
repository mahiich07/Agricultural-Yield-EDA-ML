# Agricultural Yield Analysis using EDA and Machine Learning

## Project Description

This project performs Exploratory Data Analysis (EDA) and Machine Learning on an Agricultural Yield Dataset.

The main objective of this project is to analyze different agricultural factors affecting crop yield and build a Linear Regression model to predict yield.

## Dataset

The dataset contains agricultural information including:
- Rainfall
- Temperature
- Nitrogen
- Crop related features
- Soil related features
- Yield

## Tasks Performed

### Part A: Understanding the Dataset
- Loaded and explored the dataset
- Checked number of rows and columns
- Displayed column names and records
- Checked data types
- Identified missing values
- Generated descriptive statistics

### Part B: Exploratory Data Analysis (EDA)
- Created distribution plots for numerical features
- Analyzed crop type frequency
- Analyzed soil type frequency
- Studied yield distribution
- Created scatter plots
- Performed correlation analysis
- Compared average yield by crop and soil type

### Part C: Data Preparation
- Identified categorical variables
- Applied One-Hot Encoding
- Separated input features and target variable
- Prepared dataset for machine learning

### Part D: Machine Learning
- Split dataset into training and testing data
- Built a Linear Regression model
- Analyzed model coefficients and intercept
- Identified important features affecting crop yield

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
- GitHub

## Machine Learning Model

Model Used:
- Linear Regression

Target Variable:
- Yield

## Conclusion

The analysis shows that environmental and agricultural factors such as rainfall, temperature, and nutrients influence crop yield. A Linear Regression model was successfully trained to understand the relationship between input features and yield.
