# Agricultural Yield Analysis using EDA and Machine Learning

## Project Description
This project performs Exploratory Data Analysis (EDA) and Machine Learning on an Agricultural Yield Dataset.

The main objective is to understand factors affecting crop yield and build a Linear Regression model.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
The dataset contains agricultural information including:

- Crop Type
- Soil Type
- Rainfall
- Temperature
- Fertilizer Usage
- Yield per hectare


## Tasks Performed

### Part A: Understanding Dataset
- Dataset overview
- Data types analysis
- Missing value detection
- Descriptive statistics

### Part B: Exploratory Data Analysis
- Distribution analysis
- Crop type analysis
- Soil type analysis
- Yield analysis
- Scatter plots
- Correlation heatmap
- Group-based analysis

### Part C: Data Preparation
- One-Hot Encoding
- Feature Selection

### Part D: Machine Learning
- Train-Test Split
- Linear Regression Model
- Coefficient Analysis


## Machine Learning Model

Algorithm Used:

Linear Regression
